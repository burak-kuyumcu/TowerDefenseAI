About Game

The Base Core is the last stable defense node in a collapsing digital frontier. An adaptive enemy AI opens portals, studies tower placements, and changes its attack strategy from wave to wave.

Each stage represents an unstable sector with different combat modifiers. The player must build, upgrade, scan, and adapt before the Core falls.

Project: AI & Games - Learning to Play
Theme: Tower Defense AI
Student: Aydın Burak Kuyumcu
Rendering: WebGL2 + JavaScript + GLSL + HTML
Camera: 6DOF translation and rotation
Objects: Z transform showcase with 3 selectable 6DOF objects
Lights: directional light + independent spotlight
Light Control: F3 modes, WASD/QE movement, arrows rotation, +/- intensity
Shaders: full-screen Toon, Neon, and X-Ray/Scan GLSL shaders
Name Scene: C camera transition to 3D object-built name showcase
AI: adaptive wave strategy and stage-aware planning
Narrative: The Last Core Protocol, an adaptive AI siege against the final defense node

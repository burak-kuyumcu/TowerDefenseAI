Camera Mode - 6DOF
F3 until Camera 6DOF mode
W/A/S/D: Move camera on X/Z
Q/E: Move camera up/down
Arrow Left/Right: Camera yaw
Arrow Up/Down: Camera pitch
PageUp/PageDown: Camera roll
Shift: Faster movement/rotation

Object Mode - 6DOF
Select a tower, enemy, or showcase object with mouse
F3 until Object 6DOF mode
W/A/S/D: Move selected object on X/Z
Q/E: Move selected object up/down
Arrow Left/Right: Rotate yaw
Arrow Up/Down: Rotate pitch
PageUp/PageDown: Rotate roll
Shift: Faster transform
Preparation mode: Shift + Arrow Keys relocate tower on grid

Spotlight Mode
F3 until Spotlight Control mode
O: Spotlight on/off
Shift + H: Spotlight helper
W/A/S/D: Move spotlight on X/Z
Q/E: Move spotlight up/down
Arrow Keys: Aim spotlight target X/Z
PageUp/PageDown: Aim spotlight target Y
+/-: Spotlight intensity
Blue ring: Spotlight target marker

Directional Light Mode
F3 until Directional Light mode
P: Directional light on/off
Arrow Left/Right: Direction yaw
Arrow Up/Down: Direction pitch
PageUp/PageDown: Direction roll value
Alt + W/A/S/D: Move directional light on X/Z
Alt + Q/E: Move directional light up/down
+/-: Directional light intensity
Yellow arrow: Directional light direction helper

Shader & Tactical Scan
M: Change shader mode
Standard: Normal render
Toon: Full-screen cartoon shader
Neon: Full-screen glow shader
X-Ray: Full-screen tactical scan shader
X-Ray + selected tower: Show relocation hints

About Game

Tower Defense AI is a WebGL2 3D strategy game where the enemy AI adapts to your tower choices, stage modifiers, and defensive patterns.

Project: AI & Games - Learning to Play
Theme: Tower Defense AI
Student: Aydın Burak Kuyumcu
Rendering: WebGL2 + JavaScript + GLSL + HTML
Camera: 6DOF translation and rotation
Objects: selectable 3D transform showcase objects
Lights: directional light + independent spotlight
Shaders: full-screen Toon, Neon, and X-Ray/Scan GLSL shaders
Name Scene: 3D object-built name showcase
AI: adaptive wave strategy and stage-aware planning
